From spec to conformance tests in 3 commands

1. Point

Specify your OpenAPI spec and target URL.

cherenkov validate \
  --spec openapi.yaml \
  --target http://localhost:8000

2. Generate

AI generates 200+ conformance tests per endpoint.

# Covers: happy path, null fields,
# missing required, type mismatch,
# auth, pagination

3. Eject

Export vanilla Playwright tests. Zero lock-in.

cherenkov eject \
  --output ./tests
# Produces plain .spec.ts files

Get started in 5 minutes

pip install cherenkov && cherenkov doctor

Requires Python 3.10+ and Ollama (for AI generation).
