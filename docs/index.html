CHERENKOV.

API Conformance Test Generator

Spec in → Playwright tests out → zero lock-in. A local-first tool that reads your OpenAPI spec and generates pure Playwright API tests using a local 7B LLM — no cloud, no API keys, no vendor lock.

How it works

  • Point at your OpenAPI spec — INGEST parses every endpoint
  • LLM generates Playwright tests with openapi-fetch client
  • 6-gate review: syntax, structure, AST, assertions, tsc, Prism dry-run
  • Run against your real server with cherenkov validate
  • Eject to standalone Playwright — zero dependency on CHERENKOV

Quick start

pip install cherenkov
cherenkov init
cherenkov validate --spec openapi.yaml --target http://localhost:3000

Key features

  • Spec-derived expected status codes (catches 422-vs-400 bugs)
  • Suggest-only healing — never auto-edits test code
  • Human-in-the-loop review queue with confidence scoring
  • MCP server for Claude Desktop / Cursor integration
  • SARIF output for GitHub Security tab
  • Desktop app (Tauri), mobile testing (Maestro), K8s operator

Architecture

Local LLM (qwen2.5-coder:7b via Ollama) — GPU or CPU — runs entirely on your machine. No telemetry, no API calls, no data leaves your network. Enterprise tier adds SSO, RBAC, audit logs.
