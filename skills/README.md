# CHERENKOV Skills Directory

Welcome to the `skills/` directory. Inspired by the `mattpocock/skills` pattern, this directory contains highly specific, stack-aware markdown instructions for CHERENKOV autonomous agents.

## Purpose
When an autonomous agent (like the `Explorer` or `Copilot`) performs a complex task, it should first read the relevant skill file from this directory. This ensures the agent perfectly understands *how* to perform the task according to CHERENKOV's strict design invariants (e.g., the D7 suggest-only rule).

## How to use
Future agents: Before executing an E2E testing run, mapping an integration, or doing a visual audit, read the `.md` file matching your task in this directory.
