# OpenAPI Spec Integration

This guide is under active development. Check back soon or [contribute on GitHub](https://github.com/moaidmoatasem/cherenkov-qa).

## Quick Reference

CHERENKOV supports OpenAPI 3.x (YAML or JSON). Point to your spec:

```bash
cherenkov validate --spec ./api/openapi.yaml --target http://localhost:8000
```
